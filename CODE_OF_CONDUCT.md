# Code Of Conduct / 行为准则

## 中文

请在本仓库中保持尊重、建设性和协作性的交流方式。

我们希望参与者：

- 尊重不同背景与经验的人
- 聚焦问题本身，而不是攻击个人
- 以帮助项目变得更好为目标提出反馈

维护者可能会编辑、隐藏或关闭违反上述原则的内容。

如果你需要报告不当行为，请先通过维护者可见的仓库渠道发出说明；涉及安全问题时请遵循 [`SECURITY.md`](./SECURITY.md)。

## English

Please keep participation respectful, constructive, and collaborative.

We expect participants to:

- respect people with different backgrounds and experience levels
- focus on the issue, not the person
- give feedback with the goal of improving the project

Maintainers may edit, hide, or close content that violates these expectations.

If you need to report unacceptable behavior, contact the maintainers through repository channels that do not expose unnecessary private detail. For security matters, follow [`SECURITY.md`](./SECURITY.md).
