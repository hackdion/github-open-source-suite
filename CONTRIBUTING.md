# Contributing / 贡献指南

## 中文

欢迎改进这套技能。

提交前请尽量遵守以下规则：

- 先看 `docs/superpowers/specs/` 和 `docs/superpowers/plans/`，确认改动方向一致
- 如果修改共享模板或共享规范，请同时检查会不会影响多个技能
- 如果修改技能触发条件、工作流或默认值，请保持 6 个技能之间的衔接一致
- 尽量把改动做小、做清楚，不要把无关重构混进同一批提交
- 涉及用户可见行为变化时，请同步更新文档

适合用 GitHub Discussions 的内容：

- 提问
- 想法讨论
- 流程方向建议

适合用 GitHub Issues 的内容：

- 明确可复现的 bug
- 明确的功能缺口
- 可执行的改进项

## English

Contributions are welcome.

Before opening a pull request, please try to follow these rules:

- read the design and plan docs under `docs/superpowers/` first
- if you change shared templates or references, check the impact across all six skills
- if you change trigger rules, workflow boundaries, or defaults, keep the suite consistent end to end
- keep changes focused and avoid bundling unrelated refactors
- update docs when user-visible behavior changes

Good candidates for GitHub Discussions:

- questions
- design conversations
- workflow suggestions

Good candidates for GitHub Issues:

- reproducible bugs
- clear feature gaps
- actionable improvement requests
