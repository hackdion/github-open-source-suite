# Support / 支持

## 中文

本仓库默认把支持分成两类：

- GitHub Discussions：提问、使用交流、流程讨论
- GitHub Issues：可复现 bug、明确功能请求、待执行改进项

提交前建议先检查：

- `README.md`
- 已有 Discussions
- 已有 Issues

安全问题不要作为普通 issue 或 discussion 公开披露。请改用 [`SECURITY.md`](./SECURITY.md) 中的流程。

## English

This repository uses two support paths by default:

- GitHub Discussions for questions, usage help, and workflow discussions
- GitHub Issues for reproducible bugs, clear feature requests, and actionable improvements

Before opening a new thread, please check:

- `README.md`
- existing Discussions
- existing Issues

Do not disclose security issues as normal issues or discussions. Use the process in [`SECURITY.md`](./SECURITY.md).
