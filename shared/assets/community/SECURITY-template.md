# Security Policy

## Reporting A Vulnerability

Please report security issues to {{SECURITY_CONTACT}} instead of opening a public issue.

Include:

- affected version or commit
- reproduction steps
- impact summary
- any workaround if known

## Scope

Use this process for vulnerabilities that could affect confidentiality, integrity, or availability.

For general setup help or usage questions, use {{SUPPORT_CHANNEL}}.
